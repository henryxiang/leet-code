#231. Power of Two | Algorithms | Easy | Math, Bit Manipulation

Category: Algorithms
Difficulty: Easy
Tags: Math, Bit Manipulation

Given an integer, write a function to determine if it is a power of two.

Example 1:


Input: 1
Output: true 
Explanation: 20 = 1


Example 2:


Input: 16
Output: true
Explanation: 24 = 16

Example 3:


Input: 218
Output: false
